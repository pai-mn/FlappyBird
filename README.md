## FlappyBird (Java)

一个用 Java 编写的简易 Flappy Bird 小游戏示例。

### 环境要求
- **JDK**: 25 或更高版本

### 项目结构
```
.
  ├─ FlappyBird.java    // 入口与主要逻辑
  └─ README.md
```

### 运行
无需编译，直接在终端中执行：

```bash
java FlappyBird.java
```


### 操作说明
- **空格键**：控制小鸟上升
- 躲避障碍并尽量坚持更久以获得更高分数
